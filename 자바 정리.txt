클래스 : 자바 프로그램을 이루는 가장 작은 파일 단위

클래스들이 모여서 프로그램이 된다고 보면 된다.

실행용 클래스 : main 메서드가 들어있다. 프로젝트당 보통 1개가 있다.

참조용 클래스 : main메서드가 존재하지 않는다. 참조를 하기위해 만드는 클래스
ㄴ 참조 자료형(String)이라 할 수 있다.
ㄴ 설계도/설명서 : 상태,기능 이 들어있다.
ex) 붕어빵틀, 와플기계 ..

하나의 데이터 만으로는 표현하기 힘든것을 클래스로 만든다.
-> 코드로 표현을 해야하기 때문
ex)사람, 책, 마우스, 모니터 ... 현실에 존재하는 대부분의 것들

객체 : 각각의 상태와 기능을 가진다.(= 각각의 필드와 메서드를 가진다.)
클래스를 통해 만들어진 것들이 객체이고
클래스는 재사용성이 높다(1회용이 아니다.)
ex) 붕어빵, 와플 ..

다른 언어들과 쓰는 방식이 비슷함
c, java, js, python, c++,c# ...
