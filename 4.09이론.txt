파일클래스의 메서드

isFile() : 최종 목적지가 파일이면 true
isDirectory() : 최종 목적지가 폴더면 true
list() : 디렉토리 내 목록 반환
length() : 파일 크기 반환
getName() : 파일명을 반환
gerAbsolutePath() : 절대 경로 반환
exists() : 목적지가 존재하면 true
delete() : 파일이나 폴더를 삭제하고 true반환
ㄴ 폴더 내부가 비어있어야 삭제가능
createNewFile() : 새로운 파일 생성

입출력 스트림

바이트 기반의 스트림 : byte단위로 읽어온다
이미지, 음성, 영상은 byte로 이루어진 데이터라서, 문자 인코딩을 
적용하면 데이터가 손상될 수 있다.

문자 기반의 스트림
문자열을 처리하도록 설계가 되었고, 내부적으로
문자 인코딩(UTF-8 ..)처리를 한다.
