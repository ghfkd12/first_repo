HTML

<태그명> ... </태그명>

내용(content)

- 열린 태그와 닫힌 태그안에 둘러싸인 문자열

엘리먼트(element)

- 태그와 내용을 합친 형태

엘리먼트가모여 웹페이지가 만들어짐

- 자식요소들은 감싸진다

# <html> 
   <head>
      <title></title> 
   </head>
   <body>
     <h1></h1>
     <p></p>
     <div></div>
     <input>
     <br>
     <a></a>
     <hr>
     <img>
    
   </body>
</html>

html은 코드를 작성하면 위에서부터 아래로 요소들이 보여진다

속성(attribute) 

- a태그에 들어가는 href : 이동할 페이지의 링크를 지정
> 정보를 가지고 넘어갈 수 없다.

- img 태그에 들어가는 src : 이미지의 경로를 지정
                     width : 요소의 너비
                     heigth : 요소의 높이
                     alt : 이미지가 출력되지 않을 때 대신 나올 문구
                     style : 요소에 직접 디자인을 설정할 수 있다

- 시작태그 안에서 사용되며 기능 설정 사항을 지시 혹은 추가적인 정보를 제공

커스텀 태그는 존재하지 않으나 사용자 속성은 가능 : 고유한 값을 저장 할 수 있다.
> data-속성명="값" (여는태그 어딘가에 들어감)


<태그명 속성="값">내용</태그명>
> 값=value
