jsx : HTML문법을 자바스크립트 안에서 쓸 수 있도록 해주는 문법
표현식이기때문에 if, for, 매개변수로 전달, 반환할 수도 있다.

컴포넌트의 이름을 지을땐 첫글자를 대문자로 쓴다.

컴포넌트를 다 작성하면 외부에서 사용하기 위해 export를 해줘야 한다.

모듈(하나의 파일)당 하나
export default 컴포넌트명

여러개를 export하는게 가능
export {컴포넌트명, 컴포넌트명, 컴포넌트명}

컴포넌트의 호출
<컴포넌트명 />
import

dafault로 내보내것 -> 중괄호없이, 원하는 이름으로 쓸 수 있다.
그냥 내보낸것 -> 무조건 중괄호를 써야한다, 이름을 그대로 써야한다.

props
컴포넌트에서 매개변수를 의미한다.
인자는 html속성 = 값 형식으로 전달

{컴포넌트명 x = {10} y = {10} />  --> {x:10}이 매개변수 props로 넘어간다.

자바스크립트 객체
let Obj = {x:10, y:5}

value꺼내기
Obj.x -> 10
Obj.y -> 5

인라인 스타일을 줄 때는 자바스크립트 객체형식으로 준다.

const style ={
  backgroundColor = "gray",
  padding : '16px',
  borderRadius : '8px'
}

Hook
리액트에서는 데이터를 상태(state)로 관리를 한다.

상태의 생성
useState() 함수를 실행하고 나면 상태변수와 상태변수의 값을 바꿀 수 있는 배열을 반환한다.
배열을 구조분해할당으로 받는다.

const [state, setState] = useState(초기값(state));

초기값에 들어갈 수 있는 타입
기본자료형(숫자, 문자열, 논리형, null, undifined)
배열
객체
함수

스프레드연산자 (...)
반복가능 객체(배열)나 객체의 프로퍼티를 펼친다라는 의미를 가진다.
복사, 병합, 분해등의 작업을 간결하게 해주는 문법

배열에서의 사용법
const arr = [1,2,3];
const arr2 = [...arr]; -> 배열에 들어있는 [1,2,3]이 복사됨
const arr3 = [...arr,4,5] -> [1,2,3,4,5] 병합
const arr4 = [0,...arr,4] -> [0,1,2,3,4] 중간삽입 가능
